Per impostare YOCTO per la versione:
  Tera HE full
copiare il file di configurazione nella cartella di configurazione della build:
  cp 3b6-bsp/sources/meta-3b6/conf/machine/local-full.conf 3b6-bsp/build/conf/local.conf

Per impostare YOCTO per la versione:
  Tera HE half 
copiare il file di configurazione nella cartella di configurazione della build:
  cp 3b6-bsp/sources/meta-3b6/conf/machine/local-dep.conf 3b6-bsp/build/conf/local.conf
